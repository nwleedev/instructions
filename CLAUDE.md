## Git Commit

- 각 작업을 끝낼 때마다 Git에 커밋한다.
- 커밋 메시지 포맷: `<type>(<scope>): <subject>`
  - type, scope, subject는 영어
  - 커밋 본문은 한국어로 작성
  - 실제 현업 Git 커밋 컨벤션을 반영
