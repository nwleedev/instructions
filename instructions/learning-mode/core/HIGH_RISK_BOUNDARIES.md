# 학습 모드 고위험 주제 경계 검증

## 목적

이 문서는 `learning-mode`가 의료, 법률, 투자, 안전 임계 작업처럼 오답 비용이 큰 주제를 다룰 때 어디까지를 학습 지원 범위로 보고, 어디서부터는 범위를 좁히거나 사람 검토로 넘겨야 하는지 명시한다.

핵심 목표는 `고위험 주제를 아예 금지`하는 것이 아니라, `개념 학습`과 `실제 판단 대행`을 분리하는 것이다.

## 결론

- `learning-mode`는 고위험 주제에서도 개념, 용어, 공개 기준, 판단 프레임을 학습하는 용도로는 사용할 수 있다.
- 다만 개인 상황에 대한 직접 조언, 최종 결론, 행동 지시, 전문 판정 대체는 이 하네스의 단독 범위가 아니다.
- 고위험 주제에서는 항상 `범위 명시 -> 출처 기반 설명 -> 불확실성 표기 -> 사람 검토 또는 전문 상담으로 연결` 순서를 유지한다.

## 허용 범위와 금지 범위

### 1. 허용 가능한 학습 지원

아래와 같은 작업은 고위험 주제여도 `learning-mode` 범위 안으로 본다.

- 공통 개념과 용어 설명
- 공개 표준, 법령, 가이드라인의 구조적 요약
- 흔한 오개념 교정
- 판단 프레임, 체크리스트, 질문 목록 정리
- 전문가와 상담하기 전에 알아야 할 배경지식 정리

예시:

- `p-value는 무엇을 의미하고 무엇을 의미하지 않는가`
- `Scope 1/2/3 분류 기준은 무엇인가`
- `주식 가치평가에서 PER와 현금흐름이 왜 다른 신호를 주는가`
- `계약 검토 전에 확인해야 할 조항 범주는 무엇인가`

### 2. 조건부 허용

아래는 학습 지원으로 볼 수 있지만, 추가 보호 장치가 있어야 한다.

- 최신성 의존성이 큰 규정/시장/정책 설명
- 실제 사례를 바탕으로 공개 정보만 해석하는 작업
- 모델 출력이 이후 실제 결정 문서의 초안으로 재사용될 가능성이 높은 작업

추가 조건:

- 최신 공식 출처를 직접 붙인다.
- `이 설명은 일반 학습용이며 개인 상황의 최종 판단이 아니다`를 먼저 밝힌다.
- 사람이 원문 또는 원자료를 바로 확인할 수 있게 한다.
- 불확실성, 예외, 지역/기관 차이를 함께 적는다.

### 3. 단독으로 허용하지 않는 작업

아래는 `learning-mode` 단독 범위 밖으로 본다.

- 개인 대상 의료 진단, 치료 선택, 복용/중단 판단
- 개인 또는 조직의 법률 결론, 책임 판단, 계약 체결 여부 판단
- 특정 매수/매도/포지션 변경 같은 개인화된 투자 실행 조언
- 안전 임계 작업의 즉시 실행 절차를 확정적으로 지시하는 행위
- 전문 판독 자체가 핵심인 의료 이미지 해석

대응 방식:

- 설명 범위를 개념 학습으로 다시 좁힌다.
- 개인 상황 판단이 필요하면 전문가 검토 또는 사람 승인 단계가 필요하다고 명시한다.
- 필요하면 `research`로 최신 기준을 먼저 정리한 뒤, `learning-mode`는 그 기준을 이해하는 데까지만 쓴다.

## 운영 규칙

### Intake

고위험 주제 의심 신호가 보이면 아래를 먼저 판정한다.

1. 사용자가 원하는 것이 `개념 학습`인가 `내 상황의 최종 판단`인가
2. 최신 공식 기준 확인이 필요한가
3. 출력이 실제 행동이나 결정에 바로 사용될 가능성이 큰가
4. 사람 검토 없이 사용되면 피해가 커지는가

1개라도 강하게 `예`면 경계 모드로 전환한다.

### 경계 모드

경계 모드에서는 아래를 기본값으로 둔다.

1. 범위를 먼저 선언한다.
2. 개인화된 결론 대신 개념, 기준, 질문 목록을 중심으로 설명한다.
3. 공식 출처와 날짜를 붙인다.
4. 낮은 확신, 최신성 민감, 규제/기관 차이를 분리해 적는다.
5. 최종 행동 전 사람 검토 또는 전문 상담이 필요하다고 명시한다.

### 도구와 승인

- 사용자 입력이나 외부 자료를 쓰는 경우 입력 가드레일과 검토 단계를 둔다.
- MCP 도구를 쓰는 경우 승인 단계를 유지한다.
- 사람이 검토할 수 있도록 원문, 원자료, 근거 링크에 접근 가능해야 한다.

## 주제별 적용 예시

### 의료

- 허용:
  - 일반 개념 설명
  - 공개 가이드라인 구조 설명
  - 의사에게 물어볼 질문 목록 정리
- 비허용:
  - 증상만 보고 진단 확정
  - 약 복용 여부나 용량 조정 지시
  - CT/MRI 같은 전문 의료 이미지 판독

### 법률

- 허용:
  - 계약 조항 범주 설명
  - 판례/법령 체계 학습
  - 변호사 상담 전 체크리스트 정리
- 비허용:
  - 특정 사실관계에 대한 확정적 법률 결론
  - 분쟁 대응 전략의 최종 지시

### 투자

- 허용:
  - 재무지표와 위험 개념 설명
  - 공개 시장 자료 읽는 법 학습
  - 투자 논리를 검토하기 위한 질문 목록 정리
- 비허용:
  - 특정 자산의 개인 맞춤 매수/매도 판단
  - 리스크 허용도와 무관한 실행 지시

## 검증 기준

고위험 주제를 다루는 `learning-mode` 응답 또는 문서는 아래를 만족해야 한다.

- 허용 범위와 비허용 범위가 명시되어 있다.
- 개념 학습과 최종 판단 대행을 구분한다.
- 공식 출처와 최신 확인 날짜가 있다.
- 사람 검토 또는 전문 상담 필요 조건이 적혀 있다.
- 사용자 입력, 도구 사용, 민감 정보에 대한 보호 규칙이 있다.

## 남은 한계

- 현재 검증은 투자 주제와 의료 정보 읽기 주제의 `concept-only` 드라이런까지 완료했다.
- 다만 법률처럼 사실관계와 문서 책임 해석이 더 직접적인 주제는 아직 별도 드라이런이 없다.
