# 샘플 GHG 학습 로그

## Session Goal

- 오늘의 학습 목표:
  - Scope 1, Scope 2, Scope 3를 조직 경계와 운영 경계 문맥에서 구분해 설명할 수 있다.
- 최종 산출물:
  - 기관용 Scope 분류 메모 1개

## Diagnosis

- 학습자가 알고 있던 것:
  - 온실가스 배출을 직접 배출과 간접 배출로 나눌 수 있다는 정도의 직관
- 학습자가 헷갈린 것:
  - Scope 2와 Scope 3의 차이
  - 조직 경계와 운영 경계의 순서
  - 개정 중인 GHG Protocol 문서를 현재 확정 기준처럼 읽는 문제
- 난이도 판단:
  - 계산보다 기준 해석과 분류 체계 이해가 핵심인 중간 난이도

## Step Log

### Step 1

- 다룬 개념:
  - 조직 경계와 운영 경계
- 사용한 질문:
  - `스코프를 나누기 전에 먼저 무엇을 인벤토리에 넣을지 왜 정해야 하는가`
  - `시설과 자산 경계를 정하지 않으면 어떤 항목이 흔들리는가`
- 이해도 점검 결과:
  - 학습자는 처음에 항목을 바로 Scope 1/2/3으로 분류하려 했다.
- 조정 사항:
  - EPA organizational boundaries와 Corporate Standard를 근거로 `경계 -> 스코프` 순서를 먼저 고정했다.

### Step 2

- 다룬 개념:
  - Scope 1/2/3의 역할 분리
- 사용한 질문:
  - `구매 전기와 직원 통근은 둘 다 간접인데 왜 같은 스코프가 아닌가`
  - `직접 배출과 가치사슬 배출은 어떤 기준으로 갈라지는가`
- 이해도 점검 결과:
  - 학습자는 간접이면 전부 Scope 3라고 묶으려 했다.
- 조정 사항:
  - EPA scope guidance와 Scope 3 Standard를 함께 사용해 구매 전기와 가치사슬 배출을 따로 설명했다.

### Step 3

- 다룬 개념:
  - 현행 기준과 개정 진행 상황의 분리
- 사용한 질문:
  - `왜 consultation 문서를 바로 내부 운영 기준으로 쓰면 안 되는가`
  - `현재 시점 메모에는 무엇을 기준으로 쓰고 무엇을 주석으로 남겨야 하는가`
- 이해도 점검 결과:
  - 학습자는 최신 문서면 곧바로 우선한다고 생각했다.
- 조정 사항:
  - GHG Protocol public consultation page와 2025-12 progress update를 보여주며, 현행 공개 표준과 revision in progress를 분리해 쓰도록 수정했다.
