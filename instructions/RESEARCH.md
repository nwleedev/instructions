# instructions/RESEARCH.md

## 목적

외부 자료를 무조건 많이 모으는 것이 아니라,
실제 구현 또는 의사결정에 영향을 준 근거만 기록한다.

## 조사 필요 조건

다음 경우에 조사한다.

- 공식 API 사용 방식 확인
- 라이브러리/프레임워크 선택
- 최신 버전 변경 사항
- 보안/성능 관련 판단
- 표준 문서 확인

## 출처 우선순위

1. 공식 문서
2. 표준 문서
3. 실제 소스 코드
4. 보조 자료

## RESEARCH.md 기록 규칙

각 항목에는 다음을 포함한다.

- 출처
- 핵심 요지
- 우리 프로젝트 적용 방식
- 관련 결정 또는 티켓

## 금지

- 링크만 나열하는 큐레이션
- 구현에 영향이 없는 자료까지 기록
- 블로그 단독 근거로 결정
